# Bible Reading Tracker (UI)

Frontend-only implementation using Next.js 14 App Router, Tailwind, next-intl, lucide-react, framer-motion, and recharts.

## Scripts
- dev: Start dev server on http://localhost:3000
- build: Build production
- start: Start production server

## Tech
- Next.js 14 (App Router)
- Tailwind CSS
- next-intl (simple provider with localStorage toggle)
- lucide-react icons
- recharts for charts

## Structure
See `app/`, `components/`, and `locales/`.

## Notes
- Actions use console.log placeholders for backend hooks.
- Language switcher toggles between English and Amharic and persists to localStorage.
